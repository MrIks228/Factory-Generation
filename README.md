Простой Lua-скрипт для UE4SS, который принудительно заменяет генерацию мира в игре MISERY на биом Factory.

Требования:
- UE4SS

Скачать UE4SS experimental-latest:  
[https://github.com/UE4SS-RE/RE-UE4SS](https://github.com/UE4SS-RE/RE-UE4SS/releases/tag/experimental-latest)

Установка:

1. Скачайте UE4SS_v3.0.1-946-g265115c0.zip
2. Распакуйте архив в папку Win64 игры (скорее всего у вас будет C:\Program Files (x86)\Steam\steamapps\common\MISERY\MISERY\Binaries\Win64)
   У вас должны быть следующие файлы: <img width="268" height="196" alt="image" src="https://github.com/user-attachments/assets/b7dedb9f-6a0a-40fd-8e60-b5bd2c5fa033" />

4. Скопируйте папку мода в Win64\ue4ss\Mods
5. В файл mods.txt в конец (после BPModLoaderMod : 1) добавте FactoryGeneration : 1, сохраните файл

Использование:

Просто загрузите любое сохранение, скрипт зменит генерацию на фабрику. Если вы заспавнились в странном месте/под землёй, используйте F8, если не помоголо перезапустите сейв.

Важно:

- Это неофициальный мод
- Может не работать после обновлений игры
- Используйте на свой риск
